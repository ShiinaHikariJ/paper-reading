# Paper Reading Tracker

用于记录论文阅读、研究方向梳理和每周进度。

## 当前研究方向

- 大模型与小模型协同
- PHM / 故障诊断
- 数据增强
- 领域泛化
- 多标签学习
- 复合故障诊断

## 仓库结构

```text
paper-reading-tracker/
├── papers/                 # 论文总表
│   └── README.md
├── notes/                  # 单篇论文精读笔记
├── topics/                 # 按研究主题整理
├── weekly/                 # 每周进度记录
├── templates/              # 笔记模板
├── assets/images/          # 图片、论文框架图等
└── README.md
```

## 阅读进度

| 状态 | 含义 |
|---|---|
| `TODO` | 尚未开始 |
| `READING` | 正在阅读 |
| `DONE` | 已完成精读 |
| `SKIMMED` | 只做了粗读 |
| `REPRODUCING` | 正在复现 |
| `BLOCKED` | 暂时卡住 |

## 本周目标

- [ ] 阅读 2 篇数据增强相关论文
- [ ] 整理 1 篇论文精读笔记
- [ ] 更新研究问题和下一步实验计划

## 使用方式

1. 在 `papers/README.md` 中登记论文。
2. 复制 `templates/paper-note-template.md` 到 `notes/`。
3. 每周复制 `templates/weekly-template.md` 到 `weekly/`。
4. 按主题在 `topics/` 中持续整理研究脉络。
